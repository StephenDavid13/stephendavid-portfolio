## Overview

Developed a custom, responsive WordPress theme tailored for Whitecomb Range. The theme leverages PHP for dynamic functionality and Tailwind CSS for clean, modern, and mobile-friendly styling. Built with flexibility in mind to support future eCommerce expansion using WooCommerce.
The development included customised booking for their accommodation, training, and events.

## Technologies Used

- **WordPress**: Content management system for easy client updates.
- **Tailwind CSS**: Utility-first CSS framework used for efficient and consistent styling.
- **PHP**: For custom theme logic and dynamic content integration.
