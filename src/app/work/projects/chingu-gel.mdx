## Overview

Developed a responsive and customizable WordPress website for Chingu Store, utilizing the Blocksy theme framework and Tailwind CSS for a clean, modern front-end experience. The site was built to be flexible, fast, and easy for the client to maintain and expand over time.

## Technologies Used

- **WordPress** – Core content management system tailored for client use.  
- **Blocksy** – Lightweight WordPress theme framework used for modular and user-friendly design.  
- **Tailwind CSS** – Utility-first CSS framework for rapid and consistent front-end styling.
- **PHP**: For custom theme logic and dynamic content integration.
