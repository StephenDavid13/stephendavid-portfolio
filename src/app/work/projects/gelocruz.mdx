## Overview

Developed a custom WordPress website for Gelo Cruz, a Filipino solo artist, with a fully responsive and visually engaging theme. The site was built using PHP and Tailwind CSS to deliver a clean, performance-optimized front-end experience that aligns with the artist’s brand and music style. Swiper.js was integrated to create smooth, interactive sliders for showcasing media and content.

## Technologies Used

- **WordPress** – Content management system for easy updates and content publishing.  
- **Tailwind CSS** – Utility-first CSS framework used for fast, consistent, and mobile-friendly styling.  
- **PHP** – For building custom theme functionality and dynamic content integration.  
- **Swiper.js** – JavaScript library used to create responsive sliders and carousels for featured content.
