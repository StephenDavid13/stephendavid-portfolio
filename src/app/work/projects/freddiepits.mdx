## Overview

A custom WordPress theme for Freddiepits Studio, a creative photography and videography studio. Built from scratch with PHP templating and Tailwind CSS rather than a page-builder, so the visual identity stays precise and the front-end stays fast. WooCommerce was scaffolded into the theme, ready to switch on when the studio's merch arm launches.
