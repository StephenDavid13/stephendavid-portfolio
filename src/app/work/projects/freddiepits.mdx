## Overview

Designed and developed a custom, responsive WordPress theme tailored for Freddiepits Studio. The theme leverages PHP for dynamic functionality and Tailwind CSS for clean, modern, and mobile-friendly styling. Built with flexibility in mind to support future eCommerce expansion using WooCommerce.

## Technologies Used

- **WordPress**: Content management system for easy client updates.
- **WooCommerce**: Integrated to support future eCommerce capabilities.
- **Tailwind CSS**: Utility-first CSS framework used for efficient and consistent styling.
- **PHP**: For custom theme logic and dynamic content integration.
