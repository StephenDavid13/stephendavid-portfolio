## Overview

A product-finder quiz built for Fansycake, an adult toy brand running on Shopify. Visitors move through a short set of questions and land on a tailored product recommendation that links back into the store. The quiz is a separate Next.js app on Vercel, embedded alongside the Shopify theme so it can iterate on its own release cadence without touching the storefront.
